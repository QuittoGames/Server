Quitto Server

Uma base web Spring para autenticação, MCP e administração.

Este front inicial já abre pela rota raiz, conversa com o login OAuth2 e deixa o projeto pronto para evoluir para telas administrativas.

Entrar com Google Ver estrutura
Spring Boot 4
Web MVC View renderizada com Thymeleaf
Security OAuth2 + rotas protegidas
MCP Área reservada para ferramentas

Autenticação

Login social já integrado ao fluxo de segurança do Spring.

Administração

Espaço para futuras telas internas e páginas de suporte operacional.

Extensibilidade

Base pronta para novos módulos, APIs e integrações sem mudar a home.
